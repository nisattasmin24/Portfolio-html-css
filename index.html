HTML

HTML, or HyperText Markup Language, is the standard lasnguage used to create and design documents on the World Wide Web. It is a cornerstone technology of the internet, alongside CSS (Cascading Style Sheets) and JavaScript. HTML provides the basic structure of web pages by using a series of elements, which are represented by tags.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS controls the layout of multiple web pages all at once and allows for the separation of content from design, making it easier to maintain and update websites.

JavaScript

JavaScript is a versatile and powerful programming language primarily used for creating interactive and dynamic content on the web. Alongside HTML and CSS, JavaScript is one of the core technologies of web development. It allows developers to add functionality, control multimedia, animate images and create responsive user interfaces.
